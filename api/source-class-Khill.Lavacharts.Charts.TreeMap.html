  2: /**
  3:  * TreeMap Chart Class
  4:  *
  5:  * A visual representation of a data tree, where each node can have zero or more
  6:  * children, and one parent (except for the root, which has no parents). Each
  7:  * node is displayed as a rectangle, sized and colored according to values that
  8:  * you assign. Sizes and colors are valued relative to all other nodes in the
  9:  * graph. You can specify how many levels to display simultaneously, and
 10:  * optionally to display deeper levels in a hinted fashion. If a node is a leaf
 11:  * node, you can specify a size and color; if it is not a leaf, it will be
 12:  * displayed as a bounding box for leaf nodes. The default behavior is to move
 13:  * down the tree when a user left-clicks a node, and to move back up the tree
 14:  * when a user right-clicks the graph.
 15:  *
 16:  * The total size of the graph is determined by the size of the containing
 17:  * element that you insert in your page. If you have leaf nodes with names too
 18:  * long to show, the name will be truncated with an ellipsis (...).
 19:  *
 20:  *
